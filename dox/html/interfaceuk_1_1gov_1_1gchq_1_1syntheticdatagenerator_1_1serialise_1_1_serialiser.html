Detailed Description

IMPORTANT: All instances of this interface must be thread safe. That is, they must support multiple threads performing calling either Serialiser#deserialise(InputStream) or Serialiser#serialise(Stream, OutputStream) concurrently. The easiest and recommended way to do this is to make the

Serialiser

instance stateless; don't store anything related to a particular de/serialisation operation in class member fields.
